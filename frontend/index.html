This is a simplified development server for the Todo Chatbot.

🎯 Full Kubernetes Deployment

To deploy the complete governed Todo Chatbot system on your local Kubernetes cluster:

  1. Open PowerShell as Administrator
  2. Run: E:\Hackathon_2\Phase_4\complete-k8s-deployment.ps1
  3. Wait for installation and deployment to complete
  4. Access the full system at the provided URL

The full deployment includes AI operations, governance policies, and self-healing capabilities!
